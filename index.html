  • Squishmallow Store Website

    I created an e-commerce website using server-side JSON, client-side JavaScript,HTML, and CSS. Although the final website was my own work, I collaborated with a partner on the server-side JSON, such that, we had a shared project backend. In the backend, my key contributions included implementing a POST endpoint for becoming a member of the loyalty program, a GET endpoint for an FAQ page, and a POST endpoint to add items to the cart. Features I implemented in the front-end include a main view with all the webpage's products, a single-product view for the selected product, and a cart view. The main page has a sorting filter, the cart view displays all selected items, with the total prices calculated, and the single-product view includes buttons to increase/decrease quantities of the product and add it to the cart. The webpage also includes a POST endpoint for contacting the store and a pop-up for becoming a loyal customer. A YouTube video demonstrating the functionality of the webpage will be posted shortly!
